تثبيت VCG v1.0

$ git clone https://github.com/syrian-vcg/vcg-lang.git
$ cd vcg-lang && make
$ ./vcgc --version # vcgc 1.0.0
$ ./vcgc hello.vcg # ينتج hello.html
$ ./vcgc -r hello.vcg # تشغيل مباشر
$ ./vcgc -o out.html hello.vcg # اسم مخصص
$ ./vcgc --tokens hello.vcg # dump tokens
$ ./vcgc --ast hello.vcg # dump AST
