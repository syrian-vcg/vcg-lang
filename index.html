لماذا تختار vcg-lang؟

نحن لا نبني لغة جديدة لتعقيدك، بل نوفر طريقة أبسط للتعبير عن واجهات الويب والأدوات البرمجية المعقدة في ثوانٍ.

أداء نقي وبلا قيود

المترجم الخاص بـ vcg-lang ينتج كود HTML/CSS نظيفاً ومتجاوباً بالكامل من غير تكرار برمي أو تعقيدات معتادة في بيئات العمل الأخرى.

صياغة (Syntax) فائق البساطة

لقد صممنا القواعد النحوية للغة لتكون مقروءة للبشر ومفهومة للحواسيب بشكل فوري، فلا حاجة لقضاء أيام في فهم طريقة التوثيق والتعلم.

جاهزية تامة للأجهزة

كافة المخرجات التي تولدها vcg-lang تدعم الشاشات الذكية، الهواتف المحمولة، والشاشات العملاقة دون الحاجة لكتابة كود إضافي مخصص.

كيف تبدو لغة vcg-lang؟

شاهد المقارنة بين كود لغة vcg-lang وبين النتيجة البرمجية الصافية والجميلة الناتجة عنها. 

# تعريف بطاقة مستخدم تفاعلية

التوثيق الشامل

جاهز لبدء تجربتك الأولى مع vcg-lang؟

اكتشف التوثيق الشامل الذي يشرح كافة الخيارات والمكونات والسمات البرمجية لتستمتع بعملية توليد فائقة السهولة والسرعة.

ابدأ قراءة التوثيق المطور
الدليل السريع من الطرفية 
# قم بتثبيت الأداة عالمياً
$ npm install -g vcg-compiler

# أنشئ ملفك الأول
$ echo "Card MyCard { title: 'مرحباً' }" > app.vcg

# ابنِ المشروع الآن
$ vcg-compiler app.vcg -o index.html
